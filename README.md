# Arcade Portal

A self-contained browser arcade with 110 mini-games — pure HTML, CSS, and JavaScript. No build tools, no dependencies, no internet required. Just open and play.

## Play

Open `index.html` in any modern browser. No server needed — works at `file://` URLs.

## Games

110 games across 11 categories:

| Category | Count |
|---|---|
| Classic Arcade | 10 |
| Puzzle | 15 |
| Card & Board | 12 |
| Word & Trivia | 10 |
| Action & Shooter | 12 |
| Platform & Adventure | 10 |
| Strategy | 10 |
| Physics & Simulation | 10 |
| Creative & Art | 8 |
| Memory & Reflex | 8 |
| 2-Player Local | 5 |

## Features

- Search and filter by category or difficulty
- High scores saved to LocalStorage
- Keyboard + touch/swipe controls
- Works fully offline

## Tech

Pure HTML5 + CSS3 + Vanilla JS. Canvas 2D API for rendering. Zero external dependencies.
